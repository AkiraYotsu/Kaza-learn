1. HTML

1. Belajar tentang tag dan attribute

Disini aku mempelajari bahwa HTML memiliki struktur dasar yang terdiri tag dan attribute, tag sendiri merupakan penanda utama dalam web untuk memberitahu browser bagian pada halaman sementara attribute adalah informasi tambahan pada tag

1. Belajar tentang head dan body

Disini aku mempelajari bahwa ada yang namanya head sebagai kepala dan info tidak terlihat untuk web, body yang menjadi isi dari web dan tag yang merupakan fungsi perintah dan cara mengaplikasikannya di index pada HTML. Selain itu aku belajar bahwa setiap halaman web biasanya punya judul, bab, dan subbab yang diaplikasikan pada jenis heading tag yaitu:

  1. h1 = Judul
  2. h2 = Bab atau Bagian besar
  3. h3 = Subbab berupa keterangan bab atau bagian besar
  4. h4 = Sub-subbab berupa keterangan pada subbab
  5. h5 = Detail kecil
  6. h6 = Catatan super kecil

2. CSS

1. Struktur dasar

1.Bentuk perintah CSS

Aku mempelajari bahwa CSS itu terdiri dari selector, kurung kurawal, property dan value yang merupakan cara menjalankan perintah paling dasar di CSS.

No. Perintah Fungsi Contoh
1. selector Memilih target yang akan diubah h1, div, p
2. kurung kurawal Kurung yang digunakan untuk membungkus aturan {}
3. property untuk menentukan jenis perubahan color, font-size, text-align
4. value nilai yang diubah center, 0px, blue
